How use openconnect to connect to an AnyConnect VPN with Qubes
==============================================================

Configuration files
-------------------
/rw/config/openconnect/userpassword.txt
```
<LOGINNAME>
<GROUP>
<PASSWORD>
```

/rw/config/openconnect/openconnect.conf
```
<IPADRESS>
<SERVERFINGERPRINT>
<PORT>
```
